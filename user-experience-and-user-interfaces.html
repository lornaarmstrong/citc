WHAT IS UI/UX DESIGN?

The field of UI/UX design is all about the user! In UI design, you get to design the interfaces that the user will interact with - this can include working on the layout of a website, but also working on deisgning anything that allows someone to interact with a product or service. UX, on the other hand, takes into account the user's journey. You can help design solutions to problems that users are experiencing, both on and off the screen.

This can include making sketches and prototypes, getting feedback from users and clients and other designers, creating colour schemes and using design theory.
