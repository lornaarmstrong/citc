WHAT IS MACHINE LEARNING?

Machine Learning involves using algorithms that can improve over time, through their own experience. It's a bit like how human's can learn things - Machine Learning is about helping machines to learn things. It is a bit of a tricky concept to get your head around at first, but there are so many interesting applications of Machine Learning. We can teach computers how to play games, solve puzzles or perform tasks like sorting pictures of cats and dogs into the correct category.
