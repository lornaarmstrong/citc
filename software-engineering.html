WHAT IS SOFTWARE DESIGN & ENGINEERING?

Software Design and Software Engineering are subfields within Computer Science which involve programming and creating software. You use programs and software everyday and so does the world around you! Software is in hospitals, in schools, shops and in homes all over the globe. A software engineer is someone who designs, develops, maintains, tests and evaluates software. It often involves writing code, solving problems, fixing errors, writing documentation for your code, and working with other software engineers.

The main stages of software development include: design (working out what you want to design, what features it needs, who will use it etc.), development (writing the code, running it, editing the code), maintenance (making sure there are no bugs and sorting them out if any occur), testing (running tests that show which parts of the program work as expected and which don't), and finally, evaluation (looking at the code and all of its tests and seeing what is good, bad and what could be improved).
